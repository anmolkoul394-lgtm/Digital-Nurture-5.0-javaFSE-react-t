# Module 16 - Gen AI Fundamentals

The last module in this track, and a different kind of topic from everything
before it - instead of a technology you install and run, this is about
understanding generative AI conceptually, and specifically getting comfortable
using GitHub Copilot as a working developer, since that's the tool you'll actually
have running alongside you in an IDE from here on.

## How this module is different from the rest of the course

There's no `pom.xml` or running application here - most folders are a README with
real explanation, plus worked examples (actual prompts, before/after comparisons,
sample Copilot suggestions) rather than compiled code. Where a concept benefits from
seeing it applied to code, the examples reuse the Task Manager API from Week 3, so
you're not learning prompting in the abstract - you're seeing it applied to code you
already understand.

## Suggested order

Read 1 → 2 → 3 first (the underlying AI concepts), then 4 → 5 → 6 → 7 (the Copilot
specific material) - prompting and context are the ideas Copilot itself is built on,
so they make a lot more sense once you've read those first.
